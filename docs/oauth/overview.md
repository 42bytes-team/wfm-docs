# Overview

OAuth 2.0 is not publicly available yet.

Public client registration is not open, and third-party developers cannot register OAuth applications at this time. OAuth 2.0 support is still in development and will be released later.

For now, integrations that require user authorization must continue using the existing v1 authorization flow.

Stay tuned for updates. Questions and future availability notices will be shared in our Discord server:
