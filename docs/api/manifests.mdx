# Manifests And Collections

Endpoints for version metadata and public collection manifests.

## Get Versions

<ApiEndpoint
  method="GET"
  path="/v2/versions"
  summary="Get current server, app, and collection versions."
/>

Clients can poll this endpoint to detect when local cached collections should be refreshed.

### Request

#### URL Parameters

None.

### Response

#### Body

```json
{
  "apiVersion": "x.x.x",
  "data": {
    "apps": {
      "ios": "x.x.x",
      "android": "x.x.x",
      "minIos": "x.x.x",
      "minAndroid": "x.x.x"
    },
    "collections": {
      "items": "base64",
      "rivens": "base64",
      "liches": "base64",
      "sisters": "base64",
      "missions": "base64",
      "npcs": "base64",
      "locations": "base64"
    },
    "updatedAt": "2021-05-21T14:59:02Z"
  },
  "error": null
}
```

#### Data Fields

| Field | Type | Description |
|---|---|---|
| `apps` | <ApiBadge kind="neutral" label="VersionApps" /> | Current and minimum supported mobile app versions. |
| `collections` | <ApiBadge kind="neutral" label="VersionCollections" /> | Collection version hashes. |
| `updatedAt` | <ApiBadge kind="neutral" label="string" /> | Last version update timestamp. |

See [Versions](../data-models.mdx#versions).

## Get Item Set

<ApiEndpoint
  method="GET"
  path="/v2/item/{slug}/set"
  aliases={["/v2/itemId/{itemId}/set"]}
  summary="Get the requested item and related set items."
  badges={[{kind: 'i18n', icon: '🇬🇧', label: 'Language aware'}]}
/>

If the requested item is not part of a set, the response contains only that item. If the item is part of a set or is the set root, the response contains all items in that set.

### Request

#### URL Parameters

| Name | Type | Description |
|---|---|---|
| `slug` | <ApiBadge kind="neutral" label="string" /> | `slug` field from the [Item](../data-models.mdx#item) model. |
| `itemId` | <ApiBadge kind="neutral" label="string" /> | `id` field from the [Item](../data-models.mdx#item) model. |

### Response

#### Body

```ts
{
  apiVersion: "x.x.x",
  data: {
    id: "54a73e65e779893a797fff72",
    items: Item[],
  },
  error: null,
}
```

#### Data Fields

| Field | Type | Description |
|---|---|---|
| `id` | <ApiBadge kind="neutral" label="string" /> | ID of the requested item. |
| `items` | <ApiBadge kind="neutral" label="Item[]" /> | Items in the set, or only the requested item when it is not part of a set. |
