# Dashboard

Endpoints for dashboard content used by Warframe.market clients.

## Get Dashboard Showcase

<ApiEndpoint
  method="GET"
  path="/v2/dashboard/showcase"
  summary="Get mobile app main screen dashboard content with featured items."
  badges={[{kind: 'i18n', icon: '🇬🇧', label: 'Language aware'}]}

### Request

#### URL Parameters

None.

### Response

#### Body

```ts
{
  apiVersion: "x.x.x",
  data: DashboardShowcase,
  error: null,
}
```

`data` is a [DashboardShowcase](../data-models.mdx#dashboardshowcase) model.
