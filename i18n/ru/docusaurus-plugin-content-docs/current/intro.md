# Публичная документация Warframe.market

Это пример переведенной страницы для Docusaurus i18n. Он показывает, где должны жить переводы документации.

Основная документация пока пишется на английском. При миграции существующих страниц из Notion переводите только проверенное публичное поведение.

## Локальная разработка

Запуск русской версии:

```bash
yarn start -- --locale ru
```

Если перевод для страницы отсутствует, Docusaurus может использовать английскую версию в зависимости от режима сборки и настроек.
